INSTALLATION

1. Install Python 3.7 or higher
2. Install required packages: pip install telethon colorama
3. Get API credentials from my.telegram.org 
4. Edit script and set: API_ID and API_HASH
5. Run the script: python termingram.py

COMMANDS {

1. msg          - Send Message
2. fmsg         - Send File Message
3. del-msg      - Delete Message
4. fwrd-msg     - Forward Message
5. pin-msg      - Pin Message
6. unpin-msg    - Unpin Message
7. dl-msg       - Download Message
8. chats        - Show All Chats
9. me           - Your Information
10. pid         - Get Peer ID
11. member-list - List Group Members
12. ban-user    - Ban User From Group
13. admin-rights - Change Admin Rights
14. profile-pic - Download Profile Picture
15. leave       - Leave Group or Channel
16. opench      - Open & View Messages
17. exit        - Exit Program

}

OUTPUT FOLDERS

downloads/     - Downloaded media files
profile_pics/  - Downloaded profile pictures
chat_exports/  - Exported chat history

NOTES

- First run asks for phone number and verification code
- Supports Persian and Arabic text
- Session file "session_eventg.session" is created automatically
- Use @username or numeric ID for targets
